# learn-verify

Verifies that you can connect to Learn.co via the command line.

## Installation

Install with:

```
$ gem install learn-verify
```

Alternatively, add this line to your application's Gemfile:

```ruby
gem 'learn-verify'
```

And then execute:

    $ bundle

## Usage

In your code:

```ruby
require 'learn_verify'
```
